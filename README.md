# 项目简介

## 核心功能
- 基于YOLOv8的滑坡检测系统，支持滑坡目标识别
- 提供2000张标注数据集，包含'landslide'一类目标
- 支持多种输入源检测：单张图片、图片文件夹、视频文件、本地摄像头
- 界面功能包括：实时显示目标位置、目标总数、置信度信息
- 支持自定义检测参数：置信度阈值、IOU阈值
- 提供多种显示选项：显示标签、原图显示
- 支持批量检测并查看所有检测结果
- 支持检测结果保存功能

## 适用场景
- 地质灾害监测与预警
- 滑坡风险区域安全评估
- 山区环境监测
- 科研研究与教学演示

## 技术特点
- 采用Python 3.9开发，集成OpenCV、PyQt5、Ultralytics、PyTorch等技术栈
- 提供完整源码、训练好的模型及数据集
- 包含UI界面源文件和图标资源
- 配备详细的项目运行教程
- 支持PyCharm+Anaconda或VSCode+Anaconda开发环境
